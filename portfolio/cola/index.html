Coca Cola Website Project

A website made to display my personal Coca Cola (and other) cans collection. HTML, CSS and JavaScript (JQery) were used to make the website. Originally the site was hosted with a different provider with a database feature, so all images were hosted on a database, and dynamically accessed with mySQL and PHP on a dynamic website, however, now, this website is hosted on a static website host.

Small parts of Bootstrap were used, and a js plugin Lightbox was used for zoom in when a picture is clicked. All pictures were taken by myself.
